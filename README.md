# SensorFlow - Dashboard Reactivo

## Descripción
Este proyecto es una aplicación web que permite agregar, visualizar y eliminar sensores en tiempo real.  
Está desarrollada con React en el frontend y Node.js con Express en el backend.

## Tecnologías utilizadas
- React
- Node.js
- Express
- Vite
- JavaScript
- CSS

## Cómo ejecutar el proyecto

### Backend
cd server
npm run dev

### Frontend
cd client
npm run dev

Abrir en el navegador:
http://localhost:5173
